# LuckyDraw & Grouper

A React application for lucky draws and group management.

## Features
- Lucky Draw functionality
- Grouping management
- Modern UI with Tailwind/Lucide icons

## Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:
```bash
npm run dev
```
Open http://localhost:5173 to view it in the browser.

## Build

To build the project for production:
```bash
npm run build
```
The build artifacts will be stored in the `dist/` directory.

## Deployment

This project is configured to deploy to **GitHub Pages** automatically via GitHub Actions.

1. Go to **Settings** > **Pages** in your GitHub repository.
2. Select **GitHub Actions** as the source.
3. Push changes to the `main` branch to trigger deployment.
